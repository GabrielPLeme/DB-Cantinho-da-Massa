# 📚 Projeto de Banco de Dados

Este repositório contém o desenvolvimento de um projeto de Banco de Dados, abrangendo todas as etapas de modelagem e implementação do sistema.

O objetivo é aplicar os conceitos estudados na disciplina de Banco de Dados, desde a modelagem conceitual até a implementação física utilizando PostgreSQL, seguindo boas práticas de organização, padronização e documentação.

## 📂 Estrutura do Projeto

```
PROJETO
├── modelo-conceitual
├── modelo-logico
└── modelo-fisico
    ├── creates
    ├── inserts
    ├── views
    ├── functions
    ├── procedures
    ├── triggers
    └── consultas
```

## 📖 Descrição

O projeto está organizado em três etapas principais:

- **Modelo Conceitual:** representação das entidades, atributos e relacionamentos do sistema.
- **Modelo Lógico:** transformação do modelo conceitual para o modelo relacional.
- **Modelo Físico:** implementação do banco de dados em PostgreSQL, contendo scripts SQL separados por finalidade.

## 🛠️ Tecnologias Utilizadas

- PostgreSQL
- SQL
- DBML
- Draw.io (Modelagem)

## 📌 Organização

Os scripts SQL são separados por categoria para facilitar a manutenção e a execução do projeto:

- Creates
- Inserts
- Views
- Functions
- Procedures
- Triggers
- Consultas

## 🎯 Objetivo

Desenvolver um banco de dados estruturado, consistente e de fácil manutenção, aplicando conceitos de modelagem, normalização e implementação em PostgreSQL.
